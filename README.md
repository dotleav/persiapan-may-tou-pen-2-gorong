# Medical Quiz App

A single-page medical quiz application for Indonesian research methodologies.

## Key Technologies

- **TanStack Start** with React 19 — SSR-capable full-stack framework
- **TailwindCSS v4** — utility-first styling
- **TypeScript** — type-safe components
- **Netlify** — hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
